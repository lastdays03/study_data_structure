- 최단 경로 문제 등 그래프 응용 알고리즘을 학습합니다.

## 📚 핵심 학습 내용 (Ch 09 ~ Ch 12)

### 1. 그래프 (Graph) 기초 (Ch 09)
- **개념**: 정점(Vertex)과 간선(Edge)으로 이루어진 자료구조입니다.
- **종류**:
  - **무방향 그래프** vs **방향 그래프**
  - **가중치 그래프**: 간선에 비용(Weight)이 있는 그래프
- **표현 방법**:
  - **인접 행렬 (Adjacency Matrix)**: 2차원 배열 사용. 밀집 그래프에 유리. 간선 확인 O(1).
  - **인접 리스트 (Adjacency List)**: 연결 리스트(또는 리스트의 리스트) 사용. 희소 그래프에 유리. 메모리 절약.

### 2. 그래프 탐색 (Graph Traversal)
- **깊이 우선 탐색 (DFS, Depth-First Search)**
  - 한 경로로 끝까지 갔다가 돌아오는 방식
  - **스택(Stack)** 또는 **재귀** 사용
  - 미로 찾기, 위상 정렬 등에 활용
- **너비 우선 탐색 (BFS, Breadth-First Search)**
  - 시작 정점에서 가까운 정점부터 방문하는 방식
  - **큐(Queue)** 사용
  - **최단 경로** 찾기(가중치 없을 때)에 활용

### 3. 최단 경로 알고리즘
- **다익스트라 (Dijkstra)**: 하나의 시작점에서 다른 모든 정점까지의 최단 경로 (음수 간선 불가)
- **플로이드-워셜 (Floyd-Warshall)**: 모든 정점 쌍 사이의 최단 경로
