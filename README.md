## 📚 자료구조 개념 및 구현 (4주)

**대상**: 파이썬 기초 문법을 아는 학습자  
**교재**: 좌충우돌, 파이썬으로 자료구조 구현하기 (WikiDocs)  
**기간**: 4주 (1개월)  
**목표**: 파이썬을 활용한 자료구조 직접 구현 및 동작 원리 체득

### 주차별 상세 계획

**1주차: 파이썬 기초 및 선형 자료구조 (Ch 00 ~ Ch 06)**
- **학습 목표**: 파이썬의 객체 구조를 이해하고 기본적인 선형 자료구조를 직접 구현합니다.
- **핵심 내용**:
  - **Ch 00**: 자료구조의 개념과 필요성
  - **Ch 01**: 파이썬 변수는 객체에 붙인 이름표다 (참조 개념 이해)
  - **Ch 03**: 연결 리스트 (Linked List) - 노드 생성, 연결, 삽입, 삭제
  - 스택(Stack)과 큐(Queue)의 구현 원리 (리스트 활용)
- **실습**:
  - 파이썬의 `id()`를 이용한 변수 참조 확인
  - 단일 연결 리스트(Singly Linked List) 클래스 구현
  - 스택과 큐 클래스 구현 및 테스트
- **체크리스트**:
  - [ ] 파이썬의 변수와 객체 참조 방식(Call by Object Reference) 이해
  - [ ] 연결 리스트의 노드 구조 이해 및 구현
  - [ ] 스택과 큐의 동작 원리(LIFO, FIFO) 이해 및 구현

**2주차: 트리 및 계층적 자료구조 (Ch 07 ~ Ch 08)**
- **학습 목표**: 계층적 구조인 트리를 이해하고 재귀를 통해 순회 알고리즘을 구현합니다.
- **핵심 내용**:
  - **Ch 07**: 파이썬으로 트리 구현하기
    - 배열을 이용한 이진 트리 표현
    - 연결 리스트를 이용한 이진 트리 표현
    - 재귀(Recursion)를 이용한 트리 순회 (전위, 중위, 후위)
  - 이진 탐색 트리(BST)의 개념과 연산
- **실습**:
  - 이진 트리 노드 클래스 작성
  - 재귀 함수를 이용한 순회(Traversal) 함수 작성
  - 트리 시각화 예제 실습 (Tkinter 등 활용 가능 시)
- **체크리스트**:
  - [ ] 재귀 함수의 동작 원리 및 스택 프레임 이해
  - [ ] 트리의 부모-자식 관계 및 레벨, 높이 개념 이해
  - [ ] 전위, 중위, 후위 순회 구현 가능

**3주차: 그래프 및 탐색 알고리즘 (Ch 09 ~ Ch 12)**
- **학습 목표**: 복잡한 연결 관계를 표현하는 그래프를 익히고 길 찾기 문제를 해결합니다.
- **핵심 내용**:
  - **Ch 09**: 길 찾기 게임 - 그래프의 개념
  - 그래프 표현 방법 (인접 행렬 vs 인접 리스트)
  - 그래프 탐색: 깊이 우선 탐색(DFS)과 너비 우선 탐색(BFS)
  - 최단 경로 알고리즘 기초
- **실습**:
  - 그래프 클래스 구현
  - DFS와 BFS 알고리즘 구현 및 경로 탐색 테스트
  - 미로 찾기 또는 길 찾기 예제 구현
- **체크리스트**:
  - [ ] 그래프의 정점(Vertex)과 간선(Edge) 개념 이해
  - [ ] DFS와 BFS의 차이점 및 구현 방법 이해
  - [ ] 그래프를 이용한 문제 해결 능력 배양

**4주차: 정렬, 시각화 및 최종 프로젝트 (Ch 13 ~ Ch 24)**
- **학습 목표**: 정렬 알고리즘을 구현하고, 지금까지 배운 내용을 종합하여 응용 프로그램을 만듭니다.
- **핵심 내용**:
  - **Ch 13**: 정렬 알고리즘 (버블, 선택, 삽입, 퀵, 힙 정렬 등)
  - **Ch 16**: 이벤트 반응형 프로그램 만들기 (GUI 프로그래밍 기초)
  - 알고리즘 시각화 원리
  - **Ch 24**: 마치는 말 및 종합 정리
- **실습**:
  - 주요 정렬 알고리즘 직접 구현 및 성능 비교
  - 자료구조 시각화 프로그램 작성 (선택 사항)
  - 최종 프로젝트: 나만의 자료구조 라이브러리 완성
- **체크리스트**:
  - [ ] 다양한 정렬 알고리즘의 원리 및 시간 복잡도 이해
  - [ ] 자료구조를 활용한 실제 프로그램(예: 시각화 도구) 작성 경험
  - [ ] 전체 학습 내용 회고 및 정리


**최종 프로젝트**: 
- 자료구조 라이브러리 직접 구현 (배열, 리스트, 스택, 큐, 트리, 그래프, 해시 테이블)
- 각 자료구조의 시간/공간 복잡도 분석 문서 작성
- 테스트 코드 및 벤치마크 작성

## 🎯 학습 방법론

### 1. 개념 이해 중심 학습
- **이론 학습**: 각 자료구조의 동작 원리와 메모리 구조 깊이 이해
- **시각화 활용**: 자료구조의 동작 과정을 시각적으로 이해
- **비교 분석**: 유사한 자료구조 간의 차이점과 장단점 비교
- **복잡도 분석**: 모든 자료구조의 시간/공간 복잡도 상세 분석

### 2. 구현 우선 원칙
- **이론 30% + 구현 60% + 복습 10%**: 이론 학습 후 즉시 구현
- **직접 구현**: 모든 자료구조를 직접 구현 (라이브러리 사용 금지)
- **테스트 코드 작성**: 구현한 자료구조에 대한 테스트 코드 작성
- **에러 해결 경험**: 에러 발생 시 해결 과정 기록 및 분석

### 3. 복습 및 정리
- **주말 복습**: 주간 학습 내용 정리 및 약점 보완
- **구현 코드 정리**: 구현한 자료구조 코드를 주제별로 정리
- **복잡도 분석 문서화**: 각 자료구조의 복잡도 분석 문서 작성
- **스터디 그룹**: 다른 학습자와 구현 코드 공유 및 리뷰 (선택)

### 4. 심화 학습
- **최적화 연습**: 구현한 자료구조의 성능 최적화
- **벤치마킹**: 다양한 자료구조의 성능 비교 및 벤치마킹
- **실전 활용**: 구현한 자료구조를 활용한 간단한 애플리케이션 개발
- **커뮤니티 참여**: 자료구조 관련 커뮤니티 활동 및 지식 공유

## 마무리

이 학습 플랜은 **중급자**를 대상으로 자료구조의 **개념과 구현**을 **1개월 이내**에 체계적으로 학습할 수 있도록 최적화되었습니다.

**학습 개요:**
- **대상**: 파이썬 기초 문법을 아는 학습자
- **교재**: 좌충우돌, 파이썬으로 자료구조 구현하기 (WikiDocs)
- **기간**: 4주 (1개월)
- **목표**: 파이썬을 활용한 자료구조 직접 구현 및 동작 원리 체득

꾸준한 구현 연습과 복습을 통해 자료구조 역량을 키워나가시기 바랍니다.

**성공의 핵심:**
1. 매일 조금씩이라도 구현 연습 (습관화)
2. 이론과 구현의 균형 (이론 30% + 구현 60%)
3. 직접 구현 중심 학습 (라이브러리 사용 금지)
4. 주말 복습으로 개념 정착
5. 테스트 코드 작성으로 검증
6. 복잡도 분석 문서화로 이해도 향상
